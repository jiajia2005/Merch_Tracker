# Merchandise Collection Tracker

A simple web app to organize and track your merchandise collection by IP/series, with photo upload and cloud storage via Supabase.

## Setup
1. Create a Supabase account and project
2. Create the `merchandise` table and `merch-photos` storage bucket
3. Copy `config.example.js` to `config.js`
4. Add your Supabase URL and anon key to `config.js`
5. Open `index.html` in your browser or host via GitHub Pages

## Features
- Organize by IP/series
- Upload photos
- Track quantity and condition
- Add custom notes
- Access from any device
